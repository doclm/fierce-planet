<static> FiercePlanet.calculateResourceEffect(resource)
Calculates the proportion of a particular resource type, relative to the overall number of resources, then returns a log derivative (so minor variations have minimal impact). If the global variable FiercePlanet.ignoreResourceBalance is true, this calculation is ignored. If the global variable FiercePlanet.resourcesInTension is true, this calculation is further adjusted by the proximity of other resources.
